# Dashboard de Performance de Dealers

Dashboard interactivo desarrollado en Excel orientado al monitoreo de KPIs operativos y análisis de performance de dealers en operaciones gaming.

## Objetivos del proyecto

- Analizar KPIs operativos.
- Comparar métricas entre marzo y abril 2025.
- Visualizar evolución de incidentes y performance.
- Identificar top performers.
- Crear una visualización clara e interactiva en Excel.

## Herramientas utilizadas

- Excel
- Tablas dinámicas
- Segmentación de datos (Slicers)
- Dashboard interactivo
- Visualización de KPIs
